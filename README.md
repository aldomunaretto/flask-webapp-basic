# Flask Web App Tutorial

## Setup & Installtion
Make sure you have the latest version of Python installed.  
git clone \<repo-url\>  
pip install -r requirements.txt

## Running The App
python main.py

## Viewing The App
Go to http://127.0.0.1:5000
